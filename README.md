# Maimbot-doceker-all-in-boom

麦麦的docker一键启动脚本

安装：

```shell
chmod +x install.sh && ./install.sh
```

管理：

```shell
./common.sh help
 用法: ./common.sh [command]
 可用命令:
   start     - 启动所有服务
   stop      - 停止所有服务
   restart   - 重启所有服务
   status    - 查看服务状态
   update    - 更新代码/镜像并重启
   help      - 显示本帮助信息
```
